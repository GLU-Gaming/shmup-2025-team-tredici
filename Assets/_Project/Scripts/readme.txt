Om merge conflicts te voorkomen:
- Maak een mapje aan voor de assets en prefabs van iedere Dev (voornaam bijvoorbeeld)
- Alleen die developer mag aan de assets komen in die map.
- Bijv: De map Laura met daarin submapjes voor de scripts, Prefabs, en Materials

(Later kun je ook proberen om, in plaats van een naam, het per onderdeel te verdelen. bv: Een script folder met scripts gemaakt door meerdere developers.
Maar de kans is dan groter dat twee mensen in hetzelfde bestand hebben gewerkt.
